INSTAGRAM FILTER


JAVA

This is a program that takes an input image and applies different filters on it, is written for Object-Oriented Programming class. It is written in Java.
It has 4 different filters including "swapRedBlue" which displays image.png and filters it with changing red and blue colors in the picture,
"binarize" which ceates a black & white image using intensity threshold T,
"mean" which smooths the image using a W pixel by W pixel 2D sliding window streaming mean filter,
"median" which smooths the image using a W pixel by W pixel 2D sliding window streaming median filter.
